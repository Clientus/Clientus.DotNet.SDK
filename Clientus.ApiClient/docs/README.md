# Clientus .NET SDK

SDK ufficiale .NET di Clientus.

---

## Obiettivo

Realizzare una libreria .NET moderna, riutilizzabile e tipizzata per comunicare con il backend Clientus.

Lo SDK costituirà il livello di accesso ai dati per tutti i futuri progetti Clientus.

---

## Progetti della soluzione

```
Clientus.DotNet
│
├── Clientus.Core
├── Clientus.ApiClient
└── Clientus.Tools
```

---

## Utilizzo previsto

Lo SDK sarà utilizzato da:

- Clientus Desktop
- Clientus Mobile (servizi condivisi)
- Clientus AI
- Clientus Import Center
- Integrazioni esterne
- Tool interni
- Future API pubbliche
- Pacchetto NuGet ufficiale
